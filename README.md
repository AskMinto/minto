# The Minto Monorepo

Minto is a portfolio tracking and conversational financial intelligence app for Indian retail investors. This monorepo contains the mobile frontend (Expo/React Native), web frontend (Next.js), and backend (FastAPI) — all sharing a single Supabase database.
